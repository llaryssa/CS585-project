# CS585-project [![Build Status](https://travis-ci.org/llaryssa/CS585-project.svg?branch=master)](https://travis-ci.org/llaryssa/CS585-project)
Repository for class assignments of the course CS585

##Build instructions
* $ mkdir build
* $ cd build
* $ cmake ..
* $ make
* $ ./cs585 **or** ./cs585-test
